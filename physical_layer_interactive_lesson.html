Multiplexing

Sharing a single medium

Multiple signals can share one channel using two main strategies:

TDM

Time Division Multiplexing — each user gets a cyclic time slot. Everyone takes turns.

FDM

Frequency Division Multiplexing — the bandwidth is divided into frequency channels, one per user.

Data Rates & Signal Metrics

Physical constraints determine how much data a channel can carry. Two theorems define the ceiling.

Key concepts

Bandwidth (W)

Range of frequencies the channel can transmit. Higher W = more capacity.

Valence (V)

Number of distinct signal states. Each interval carries Q = log₂V bits.

Symbol Rate

State changes per second. Nyquist max: 2 × W symbols/sec.

S/N Ratio

Signal-to-noise ratio. Higher S/N means more reliable transmission.

Shannon's Theorem — interactive calculator

The theoretical maximum bitrate for a noisy channel:

D_max = W × log₂(1 + S/N)

Transmission Media & Devices

Data travels through physical media — wired or wireless — and hardware at Layer 1 connects everything.

Wired media
Twisted Pair

UTP (unshielded) and STP (shielded) categories. Uses RJ45 connectors. Common for LANs.

Coaxial Cable

Uses BNC connectors. Low interference. Used in older networks and cable TV.

Optical Fiber

Light pulses. Very high bandwidth, long distances, minimal interference. Ideal for backbone networks.

Wireless media
Radio Waves

Wi-Fi uses electromagnetic radio waves. 802.11ax (Wi-Fi 6) is a current standard.

Laser

Line-of-sight transmission using laser beams. High bandwidth but requires clear path.

Interconnection devices

🖧

NIC

Network Interface Card. Connects a device to the network. Identified by a unique 6-byte MAC address.

📡

Hub

Broadcasts signals from one station to all others. Acts as a signal repeater/amplifier.

🔌

Modem

Modulator-demodulator. Converts digital data to analog for transmission over analog networks, and back.
